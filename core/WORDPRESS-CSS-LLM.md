# WORDPRESS-CSS-LLM.md

## 1. Purpose

This document represents the visual representation layer of the WordPress core.

## 2. Scope

- **Included**: Core design tokens (CSS variables), global style realizations, and block-style classes.
- **Excluded**: Execution logic, data structures, and layout templates.

## 3. Core Constructs

- **Symbols**: CSS Custom Properties (`--wp--preset--color--*`).
- **Registrations**: Block style realizations.
- **Patterns**: Design systems.

## 4. Rules

- **NON-EXECUTABLE**: CSS does not possessing events.
- **REFLECTIVE**: Reflects the JSON configuration layer.
