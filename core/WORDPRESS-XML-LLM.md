# WORDPRESS-XML-LLM.md

## 1. Purpose

This document represents the data portability and legacy interface layer of the WordPress core.

## 2. Scope

- **Included**: WXR (WordPress eXtended RSS), XML-RPC, and legacy data structures.
- **Excluded**: Logic, execution triggers, and styling.

## 3. Core Constructs

- **Symbols**: XML namespace identifiers (e.g., `wp:post_type`).
- **Registrations**: Content type mapping in XML.
- **Patterns**: Data portability schemas.

## 4. Rules

- **NON-EXECUTABLE**: XML is a data serialisation format.
- Mapping only: Defines how data is represented for export/import.
