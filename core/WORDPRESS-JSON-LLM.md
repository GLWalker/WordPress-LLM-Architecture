# WORDPRESS-JSON-LLM.md

## 1. Purpose

This document represents the declarative configuration layer of the WordPress core.

## 2. Scope

- **Included**: `theme.json` schema, core presets, and block configuration metadata.
- **Excluded**: Logic, execution triggers, and styling.

## 3. Core Constructs

- **Symbols**: Key identifiers (e.g., `typography.fontSizes`).
- **Registrations**: Declarative presets (e.g., palette slugs).
- **Patterns**: Configuration structures (e.g., spacing scales).

## 4. Rules

- **NON-EXECUTABLE**: JSON is data, not logic.
- Mapping only: Defines placeholders that PHP and CSS resolve.
